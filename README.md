## Installation

```
& .\buildPlugin.ps1
& .\runVisualStudio.ps1
```

After building, copy LibGit2Sharp.dll and git2-a418d9d.dll into devenv folder (e.g., `C:\Program Files\Microsoft Visual Studio\2022\Community\Common7\IDE`).
